# Previsão de Preço de Imóveis 🏠

Este projeto realiza previsão de preços de imóveis utilizando regressão linear com Scikit-learn.

## 🔧 Ferramentas
- Python
- Pandas, Matplotlib, Seaborn
- Scikit-learn
- Streamlit (opcional)

## 📈 Resultados
- R²: 0.84
- MSE: 25849300.25
